# Lili și Copiii - Magazin Online Haine Copii Handmade

Un magazin online modern și sustenabil dedicat celor mici, creat cu dragoste și atenție la detalii.

## 🌸 Despre Proiect

Lili și Copiii este un brand românesc de hăinuțe handmade pentru copii, cu focus pe:
- Produse 100% handmade create cu iubire
- Materiale sustenabile și organice
- Design modern și confortabil
- Experiență de cumpărături online plăcută

## 🛠 Tehnologii Utilizate

- **Next.js 16** - React framework cu App Router
- **TypeScript** - Tipizare statică pentru cod robust
- **Tailwind CSS** - Styling modern și responsive
- **Lucide React** - Iconițe frumoase și consistente

## 🚀 Funcționalități

### ✅ Implementate
- Design responsive complet
- Header cu navigare multi-nivel
- Homepage atrăgătoare cu hero section
- Pagini de categorii cu filtre avansate

## 🚦 Cum să rulezi proiectul

1. Navighează în directorul proiectului:
```bash
cd lili-si-copiii
```

2. Instalează dependențele:
```bash
npm install
```

3. Rulează server-ul de dezvoltare:
```bash
npm run dev
```

4. Deschide [http://localhost:3000](http://localhost:3000) în browser.

## 📱 Responsivitate

Proiectul este complet responsive și optimizat pentru:
- Mobile (< 768px)
- Tablet (768px - 1024px)
- Desktop (> 1024px)
